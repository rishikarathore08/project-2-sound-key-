# Virtual Drum Kit

A simple, interactive web-based drum kit using only HTML, CSS, and JavaScript.

## Features

- Press `A`, `S`, `D`, `F`, `G`, `H`, `J` to play different drum sounds.
- Responsive layout for mobile, tablet, and desktop.
- Visual feedback on key press.

## Technologies Used

- HTML5
- CSS3 (No frameworks)
- JavaScript
